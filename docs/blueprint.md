# **App Name**: Nexus Alumni

## Core Features:

- Student/Professor Authentication: Secure signup/login page for both students and professors.
- Profile Management: Users can manage their profiles, including updating personal information and contact details. Students can enter major and graduation year; professors can enter department and research interests.
- Alumni Directory: A searchable directory of alumni, allowing users to find and connect with other alumni based on various criteria.  Option to filter directory by student or professor.
- Events Calendar: A calendar displaying upcoming alumni events, such as reunions, career fairs, and networking opportunities.
- Job Board: A job board where alumni can post and search for job openings.
- Mentorship Program: Facilitate mentorship connections between alumni and current students based on common interests and career goals, allowing for direct messaging within the app.
- AI-Powered Recommendation Tool: The app can use AI to recommend relevant events, job opportunities, and mentorship connections based on user profile information, indicated preferences, and network activity. The LLM will be provided access to user profile and engagement data, then will decide when and if to include each piece of information in the app.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), symbolizing trust and reliability, creating a professional yet approachable aesthetic.
- Background color: Light gray (#F5F5F5), providing a clean, neutral backdrop that ensures readability and focuses attention on the content.
- Accent color: Muted teal (#4DB6AC), used sparingly to highlight key interactive elements and calls to action, adding a touch of sophistication without overwhelming the user.
- Headline font: 'Space Grotesk', a modern sans-serif font providing a computerized, techy look.
- Body font: 'Inter', for a neutral, modern style that is highly readable.
- Simple, outlined icons to represent different sections and actions.
- Clean and modern layout with a focus on readability and ease of navigation.
- Subtle transitions and animations to provide feedback and enhance the user experience.
